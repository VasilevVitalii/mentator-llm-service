В форме /chat по аналогии с модальной формой "format response" нужно сделать форму "options llm". Тут будет настраиваться тот раздел промта, который уходит в разделе "options" (температура и т.д.). Хранится должны в localStorage с key="options".
В поле memo - json (default-ы взять из src/config.ts):
```JSON
{
    "temperature": 0,
    ...
}
```
Кнопка check отправляет на проверку на сервис (сделать API post/checkoptions), проверяется: валидность JSON-а, если каких-то парпаметров нет (например, пользователь удалил ранее) - то добавить из в JSON (это тоже считаем как валидный результат). URL ссылка внизу должно называться как-то иначе и вести на какой-нибудь внешний ресурс (страницу) - где про это параметры рассказываются подробнее.

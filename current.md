В форме /chat хочу сделать рабочими кнопки LOAD и SAVE. Это сохранение промта + настроек и их загрузка.
Формат файла хранения обеспечивается пакетом "vv-ai-promt-store". Файл в итоге будет выглядеть так:

$$begin
$$segment=model
имя модели
$$segment=format
JSON с форматом
$$segment=options
JSON с options
$$system
системное сообщение
$$user
пользовательское сообщение
$$end

Т.к. формат может содержать в файле более одного такого раздела, то
загружается первый.
При сохранении имя файла назначается так:
1. Если файл был загружен И там был один таеой раздел - то сохраняется с таким же именем файла
2. В противном случае имя файла - "promt-XXX", где XXX - текущее время в таком же формате, как при сохрании в файл респонса.

Если при загрузке модель не найдена, она не меняется и остается та, что выбрана на текущий момент.
